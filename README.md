# Instinct MO

**Educational Replica of the Kolbe A™ Index** (Fixed & Improved)

A beautiful, standalone single-file HTML web app that simulates the Kolbe A Index for learning and informational purposes.

> **Important Disclaimer**: This is **not** the official Kolbe A Index. It is an unofficial educational tool created to help people understand the concept of conative strengths and the four Action Modes. For the real, validated assessment, please visit [kolbe.com](https://www.kolbe.com/kolbe-a-index/).

## What's Fixed
- Start buttons now use modern event listeners (more reliable)
- Improved resume logic with clearer confirm dialog
- Added console logging for easier debugging
- Minor styling polish on selected options

## Features
- 36-question long-form quiz (9 questions per Action Mode)
- Instant results with your 4-number MO code
- Detailed breakdown with Initiate/Respond/Resist labels
- Mobile-friendly and beautiful design
- Progress saving in browser (resume anytime)
- Keyboard support
